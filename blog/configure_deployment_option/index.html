Configure Deployment Option

September 13, 2018 Â· 1 minute read

Begin by selecting the “Choose Source” (Configure required settings) option

Next select the correct source. I’m using GitHub, so that is the option I will choose:

The next screen will prompt you to enter your GitHub account, project, branch, etc.

Once you have made your configuration choices, select “Save” at the bottom of the screen.

That’s it!

Now, any time new code is pushed to your repository, Azure will automatically deploy the changes for you.
